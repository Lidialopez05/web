Qué es una red LAN

La red LAN (Local Area Network) es una red que conecta uno o más ordenadores dentro de un ámbito pequeño y limitado.

Qué es una red WAN

Una red de área amplia (WAN) es una gran red de ordenadores que conecta grupos de ordenadores a grandes distancias.

Qué es Internet

Internet se podría definir como una red global de redes de ordenadores cuya finalidad es permitir el intercambio libre de información entre todos sus usuarios.

Qué es la Web

La web es uno de los servicios que funcionan sobre internet, nos permite acceder a información que está ubicada en otros computadores en otros lugares del mundo.

Qué es WWW

Es un sistema interconectado de páginas web públicas accesibles a través de Internet.

Qué es el protocolo HTTP y HTTPS

HTTP (Hypertext Transfer Protocol) es el protocolo fundamental de comunicación en la World Wide Web. Su propósito principal es permitir la transferencia de información, como texto, gráficos, sonido, vídeo y otros archivos multimedia, entre un cliente (por lo general, un navegador web) y un servidor.

Página web estática vs página web dinámica

Una página estática, también conocida como plana, es el portal web que muestra contenido permanente, fijo, que no se actualiza. Allí el usuario se encuentra limitado a leer u obtener información, sin posibilidad de interacción. A diferencia de las estáticas, las páginas dinámicas son las que contienen ciertas funcionalidades para ofrecer interactividad con el usuario. Pueden ser creadas por CMS, aplicaciones externas que otorgan la posibilidad de incluir temas e instalar plugins para ampliar las herramientas de interacción con quien navegue.

Proveedor de servicios de internet. Ejemplos

El proveedor de servicios de internet es la empresa que brinda conexión a Internet a sus clientes. EJEMPLOS: Movistar, Vodafone, Orange, Yoigo, Euskaltel, Telecable, Adamo Telecom.

CMS. Ejemplos. Ventajas e inconvenientes

Los CMS son herramientas que ayudan a crear sitios web con mayor facilidad. Las ventajas que presentan son, un gran ahorro en el tiempo que toma construir un sistema o aplicación, como también una mayor facilidad de codificación y la presencia de un esquema de trabajo funcional ya establecido, lo que permite que los desarrolladores menos experimentados no tengan que lidiar con la dificultad de tener que crear su propia metodología de trabajo ni tampoco el motor del sistema, ya que pueden utilizar los que les brinda el CMS. Como ventajas existen varias, no es extraño que los sitios web creados con CMS de uso masivo presenten problemas de seguridad constantes (infecciones de virus, robo de contraseñas y eliminación de contenido, entre otros). Esto se debe a que son un blanco de ataque muy atractivo para hackers, quienes al vulnerar el motor en el que están hechos consiguen vulnerar todos los sitios web que lo utilicen. EJEMPLOS: WordPress, Drupal, Magento, Joomla.

Dirección IP. Diferencia entre una dirección IP privada y pública

Una dirección IP es una cadena de números separados por puntos. La dirección IP privada de un sistema es la dirección IP utilizada para comunicar en el interior de la misma red. Utilizando IP privadas se puede enviar o recibir información en la misma red. La dirección IP pública de un sistema es la dirección IP utilizada para comunicar en el exterior de la red. La dirección IP pública es asignada sobre todo por el ISP (Proveedor de servicios de Internet).

¿Qué es alojamiento web/hosting? Ejemplos

El hosting o alojamiento web es un servicio digital que tiene como función almacenar la información y contenido de un sitio web en centros de procesamiento de datos seguros y veloces. EJEMPLOS: CMS hub, Google Cloud,Wix, AWS.
